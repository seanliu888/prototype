# eSIM集成商系统 - 1

## 【管理端】
## 一、需求概要描述

### （一）集成商管理（Integrators）

1. **集成商多条件筛选**
   - 支持通过**集成商名称/编号、状态（正常/停用/待审核）、创建时间区间、联系人邮箱/手机** 组合筛选。

2. **集成商列表展示**
   - 以列表形式展示集成商核心信息：名称、编号、状态、联系人、账户余额、创建时间，并提供「查看详情」「编辑」「停用/启用」等操作入口。

3. **新增/编辑集成商**
   - **新增**：支持录入企业名称、联系人姓名、邮箱、手机、结算方式（预付/后付）、备注等；创建后自动生成平台账号（或由管理员指定初始密码/发送激活链接）。
   - **编辑**：支持修改集成商基本信息、状态（启用/停用）、结算方式及备注。

4. **集成商详情**
   - 进入单个集成商详情页，可查看其完整档案。

---

### （二）价格与供应商配置（Pricing & Suppliers）

1. **供应商与套餐同步（管理端）**
   - **供应商维护**：支持维护供应商信息（名称、编码、API 配置等），支持按供应商名称、状态筛选与列表展示。
   - **套餐同步**：支持通过**供应商 API** 将供应商的 eSIM 套餐同步到管理端；支持手动触发同步或配置同步周期，同步结果在管理端形成「供应商 eSIM 套餐目录」。
   - **同步日志与异常**：支持按供应商、时间范围查看同步记录，展示同步时间、成功/失败、失败原因；支持对失败项重试，关键失败可纳入告警或待办。
   - **供应商套餐目录（只读）**：管理端可查看已同步的套餐列表，支持按供应商、地区、套餐名称、数据类型、流量规格、有效期等筛选；列表展示供应商、套餐ID、套餐名称、地区、流量规格、有效期、供应商报价、最后同步时间、状态等。管理员不可在目录内直接编辑/删除，以供应商 API 同步为准。

2. **集成商专属定价**
   - 支持按集成商维度，从管理端已同步的**供应商 eSIM 套餐**中为其配置套餐售价、成本价（可选）、建议零售价等；支持批量导入/导出某集成商的价格表。
   - 支持为不同集成商配置不同供应商或套餐可见范围（白名单），实现差异化供应策略；集成商侧可见的套餐与报价来源于本管理端已同步的套餐及此处配置。

3. **配置筛选与展示**
   - 支持按**集成商、地区、套餐名称、数据类型** 筛选已配置的套餐与价格，以列表形式展示集成商、套餐、售价、成本、生效时间等，并提供「编辑」「批量调整」入口。

4. **全局价格模板（可选）**
   - 支持维护平台级「默认价格/模板」，新接入集成商时可一键应用模板再微调，减少重复配置。

---
### （三）财务与结算（Billing & Settlement）

1. **交易记录与调账**
   - 支持按**集成商、交易类型、时间区间、交易 ID** 筛选交易记录；列表展示交易 ID、集成商、金额、类型、余额快照、时间、备注。
   - 管理员可执行**人工调账**（如补偿、扣减、备注）。

---

### （四）订单与用量（Orders & Usage）

1. **订单筛选**
   - 支持通过**订单号、集成商、订单状态、创建时间区间、金额区间** 筛选全平台或指定集成商的订单，提供「重置」「收起筛选」功能。

2. **订单列表与详情**
   - 以列表形式展示订单号、所属集成商、eSIM 数量、金额、状态、创建时间等；支持点击进入订单详情，查看关联的 eSIM 列表及状态，便于客诉与对账。

---

### （五）开发者与 API（Developer & API）

1. **密钥与回调监管**
   - 管理员可查看或修改各集成商的 API 密钥状态、Webhooks 配置；在风控或客诉场景下支持「强制重置密钥」或「临时禁用 API」。

2. **开放接口功能**
   - 查询套餐：请求提供所有可用数据包的列表。可选择按国家或地区过滤。
   - 订阅套餐：单独或批量订购配置套餐。
   - 退订套餐：取消未激活的esim套餐（需供应商支持）。
   - 查询订单：支持分页查询过往订单，获取订单详情，获取esim状态、激活码等信息。

3. **开发文档**
   - 提供「API 版本与文档」说明。

## 二、需求详细描述

### （一）集成商管理（Integrators）

#### 1. 集成商列表页

**页面入口**：左侧导航「集成商管理」

**筛选条件**：
| 字段 | 类型 | 说明 |
|------|------|------|
| 集成商名称/编号 | 文本输入 | 模糊匹配名称或精确匹配编号 |
| 状态 | 下拉单选 | 全部 / 正常 / 停用 / 待审核 |
| 创建时间起 | 日期选择 | 起始日期 |
| 创建时间止 | 日期选择 | 截止日期 |
| 联系人邮箱/手机 | 文本输入 | 模糊匹配 |

**列表字段**：
| 字段 | 说明 |
|------|------|
| 名称 | 企业名称（带头像首字） |
| 编号 | 平台分配的唯一编号，如 INT-2025001 |
| 状态 | 正常（绿色）/ 停用（红色）/ 待审核（黄色） |
| 联系人 | 联系人姓名 + 邮箱/手机（脱敏显示） |
| 结算方式 | 预付（蓝色徽章）/ 后付（青色徽章） |
| 账户余额 | 当前账户余额（$） |
| 信用额度 | 信用额度金额或「不限制」 |
| 创建时间 | 创建时间 |

**操作按钮**：
- **查看详情**：跳转至集成商详情页
- **编辑**：跳转至编辑表单页
- **停用/启用**：切换集成商状态（停用后 API 不可用）
- **审核通过**：仅待审核状态显示

**页面操作**：
- 右上角「新增集成商」按钮跳转至新增表单页
- 支持分页展示

#### 2. 新增/编辑集成商页

**页面入口**：列表页「新增集成商」或「编辑」按钮

**表单字段**：

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| 企业名称 | 文本输入 | 是 | 集成商企业名称 |
| 联系人姓名 | 文本输入 | 是 | 主要联系人 |

| 信用额度 | 数字输入 | 否 | 留空表示不限制 |
| 状态 | 下拉单选 | 编辑时 | 正常 / 停用 / 待审核 |
| 备注 | 多行文本 | 否 | 备注信息 |

**业务规则**：
- 新增时自动生成平台账号或发送激活链接
- 编辑时编号不可修改
- 状态字段仅编辑时可见

**页面入口**：列表页「查看详情」按钮

**基本信息区**：

**操作日志区**：
- 预留：展示管理操作记录（何人何时执行了何种操作）

**页面操作**：
- 右上角「编辑」按钮跳转至编辑页

#### 9. 批量导入定价页

**页面入口**：定价列表「批量导入」按钮

**操作流程**：

**第一步：选择套餐**
- 筛选条件：供应商、类型、国家/地区（多选）、套餐名称、运营商（多选）
- 套餐列表：支持勾选，显示套餐Code、套餐名称、类型、国家/覆盖范围、运营商、供应商报价
- 批量操作：全选当前结果、全选本页、取消本页、取消全选
- 支持分页

**第二步：选择集成商**
- 下拉选择目标集成商

**第三步：设置定价**

**方式一：应用定价模板**
- 售价规则：按利润率（报价 + X%）或按固定金额（报价 + $X）
- 建议零售价规则：按利润率或按固定金额
- 存在时替换：勾选后覆盖已存在记录，不勾选则跳过
- 「应用模板并导入」按钮

**方式二：上传 CSV 文件**
- 「导出已选套餐为 CSV 模板」按钮
- CSV 模板字段：产品Code、产品类型、套餐名称、类型、国家/覆盖范围、运营商、流量、供应商报价、售价、建议零售价、状态
- 上传 CSV 文件后显示解析预览（最多20条）
- 存在时替换选项
- 「确认导入」按钮

**导入结果**：
- 显示导入成功/失败数量
- 「返回定价列表」按钮

#### 3. API 版本与文档页

**页面入口**：开发者与 API 页「API 版本与文档」按钮

**页面结构**：
- 顶部导航栏：返回按钮、版本徽章（v1.0）
- 左侧文档目录：可折叠的 API 分类导航
- 右侧文档内容：API 详细说明

**开放 API 功能列表**：
| 接口 | 功能说明 |
|------|----------|
| Get All Data Packages | 获取所有可用数据套餐列表，支持按地区/国家筛选 |
| Order Profiles | 订购 eSIM Profile，支持批量订购 |
| Query All Allocated Profiles | 分页查询已分配的 Profile 列表 |
| Cancel Profile | 取消/作废指定的 Profile |
| Balance Query | 查询指定 Profile 的流量余额 |
| Top Up | 为指定 Profile 购买加油包/充值流量 |
| Set Webhook | 设置 Webhook 回调地址 |
| Supported Regions | 获取支持的地区和国家列表 |

**Webhook 事件**：
| 事件 | 说明 |
|------|------|
| Profile Status | Profile 状态变更通知（激活、过期、流量用尽） |
| Balance Alert | 流量余额低于阈值时的预警通知 |
